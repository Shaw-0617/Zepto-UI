该项目为zeptoUI的一个实际应用项目。

zeptoUI在github的地址：https://github.com/Shaw-0617/Zepto-UI

zeptoUI尽量只保留最基础的js改变DOM，所需样式在实际应用中仍需自行编写。

该手机版项目依赖zepto.js,以及配套的样式。在项目中引用libs/zeptoUI下的相应文件（libs/zeptoUI/css/zeptoUi.css 以及 libs/zeptoUI/js/zepto.js ）。libs文件下的其余文件均为开源插件,按需引用即可。

部分组件使用

1.侧边栏[aside]

项目公用的侧边,点击头部的含有side类名的元素展开,点击遮罩收起,在展开时页面不可上下滑动。

可设置左滑或者右滑。

/aside.html

2.手风琴[accordion]

主要应用于侧边导航的分类,与侧边栏[aside]组合使用,实现可展开收起子分类的侧滑导航。

/accordion.html

3.回到顶部[top_box]

页面已经在顶部的时候不显示回到顶部按钮,页面滚动一定距离后出现该按钮,滚动具有动画效果。

/scroll_top.html

4.仿今日头条导航栏[nowrap_nav]

依赖iscroll,见column_1.html。

/nowrap_nav.html

5.向下展开式导航栏[drop_down_nav]

通过比较子容器和父容器的height值的大小来判断分类是否溢出。若溢出则显示向下展开箭头,不溢出则不显示向下展开箭头,点击该按钮可切换展开收起,点击遮罩可直接收起导航栏。见column_1.html,以及column_2.html。在导航栏展开时,不可滚动页面。

/drop_down_nav.html

6.标签切换[tab_box]

依赖zepto.js以及swiper.js,见首页新闻资讯部分。

/tab.html
